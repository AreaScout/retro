Retroarch

A Limadriver port of RetroArch

Download this project as a .zip file Download this project as a tar.gz file

Welcome to RetroArch port for Limadriver.

To download, install pre-requirements and compile RetroArch on our mali device please follow this steps:

$ git clone https://github.com/AreaScout/RetroArch.git
$ cd RetroArch
$ ./configure --disable-x11 --disable-gles --enable-alsa --disable-al --disable-oss --disable-jack --disable-pulse --disable-kms --disable-vg --enable-lima --enable-freetype --enable-neon --enable-floathard
$ make
$ sudo make install
